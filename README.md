# Gauteng Province Government - HR Onboarding Portal

Angular 17 frontend prototype for employee onboarding management.

## Installation
npm install

## Development Server
npm start

Navigate to http://localhost:4200

## Features
- Dashboard with onboarding statistics
- Employee management (create, view, list)
- Document upload and verification
- Onboarding checklist tracking
- Mock data for demonstration

## Tech Stack
- Angular 17
- PrimeNG 17
- TypeScript 5
- SCSS
